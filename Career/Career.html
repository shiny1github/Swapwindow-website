UI/UX Design


UI (User Interface) deals with the application's graphical layout,
which includes buttons, screen layout, animations, transitions, micro-interactions,
and so on. In short, UI is all about how things look.
UX (User Experience) design deals with how users interact with the system.
Logical navigation and how smooth and intuitive the experience
is all fall under UX design. In short, this type of design helps users have a positive experience.
